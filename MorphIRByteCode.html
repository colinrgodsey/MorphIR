Can store objects on the stack and reference their location.

Definitions:

  1. VREF - a value or value at register, symbol
  2. REF - reference, register or offset
  3. VAL - a value
  4. x32 - 32 bit, etc....
  5. TYPE - value type
  6. value - value register

Values:

  1. $# - register
  2. [a-z]+ - symbol literal
  3. %# - symbol ref

Types:

  1. i32 - 32bit signed int
  2. u32 - 32bit unsigned
  3. n - numeric value
  4. ref - native sized reference
  5. s8 - 8-bit char string
  6. s16 - 16-bit wchar string
  7. ixx - unsigned int, native bit width
  8. f32, f64, u16, i16, u8, i8
